Objectif
========

Les vieux grimoires de Merlin sont stockés dans son bureau, dans la 
bibliothèque. Vous devez afficher une liste (exploitable) des 
grimoires qui contiennent l'incantation "pq" (ou "Pq", ou "pQ", ou 
"PQ").

Certain grimoires sont cadenassés et vous n'avez pas la permission 
de les consulter. Il faut les ignorer pour l'affichage...

NOTE: seule la liste des grimoires (noms de fichiers) est demandée.
La partie de l'incantation ne doit pas être affichée.


Commandes utiles
================

Chaque commande peut écrire sur l'écran de deux manières :
  - affichage du résultat ("stdout")
  - affichage des message d'erreur ("stderr")
Par défaut, on ne peut pas différencier ces affichages.

On peut par contre rediriger les deux affichages indépendamment.

grep CHAINE FICHIER1 ... FICHIERn
  Recherche une chaine dans les fichiers et affiche les lignes 
  correspondantes.

  Options utiles :
    -l : n'affiche pas les lignes mais seulement le nom des fichiers
         qui contiennent la chaine,
    -i : ne différencie pas les majuscules et minuscules.

COMMANDE  >  FICHIER
  Envoie le résultat (stdout) de la commande dans le fichier.
  (Le fichier est écrasé.)

COMMANDE  2>  FICHIER
  Envoie les messages d'erreurs (stdout) de la commande dans le 
  fichier.
  (Le fichier est écrasé.)

/dev/null
  Fichier "virtuel" qui fait office de poubelle : tout ce qui y est 
  envoyé est ignoré.
