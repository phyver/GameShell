Objectif
========

Allez dans la cave du château.


Objectif secondaire
-------------------

Faite la différence entre ``cd -`` et ``cd ..``.


Commandes utiles
================

cd -
  Retourne dans le lieu précédent.

cd ..
  Se déplace dans le dossier parent, càd le lieu qui se trouve 
  "avant" le lieu actuel dans la liste donnée par ``pwd``.

pwd
  Affiche la position actuelle.
