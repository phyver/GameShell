Objectif
========

Supprimez toutes les araignées qui se cachent dans la cave, sans 
toucher aux salamandre porte bonheur.


Commandes utiles
================

*
?
  Le caractère "*" est un "joker" pour n'importe quelle chaine de 
  caractères.

  Le caractère "?" est un "joker" pour n'importe caractère.

  On peut les utiliser pour dénoter une liste de fichiers / dossiers 
  existants.

  Par exemple, si le répertoire contient
      fichier-1 Dossier-1 fichier-14 lustucru
  alors
    *     -->  fichier-1 Dossier-1 fichier-14 lustucru
    *1    -->  fichier-1 Dossier-1
    *s*   -->  Dossier-1 lustucru
    x*    -->  erreur, aucun fichier ne correspond
    *-?   -->  fichier-1 Dossier-1
    *-??  -->  fichier-14

  NOTE : "*" et "?" ne voient pas les fichiers cachés, il faut 
  ajouter un point explicite au début du motif.
