Objectif
========

Il parait que le roi garde le code secret de son coffre-fort inscrit 
sur un bout de papier dans sa chambre.
Trouvez ce code secret.

NOTE : le roi a probablement essayé de rendre ce papier illisible ! 


Commandes utiles
================

ls -l
  Affiche la liste des fichiers avec leurs métadonnées importantes.

  Les droits d'accès sont donnés en début de ligne.

chmod [OPTIONS] FICHIER
  Modifie les droits d'accès à un fichier ou répertoire.

  Consultez la page de manuel pour découvrir quelles options vous 
  pouvez utiliser.
