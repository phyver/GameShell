Obiettivi missione
====================

Un anziano eremita smemorato di nome Servillus si è accampato in una 
grotta con il suo vecchio libro delle pozioni di cuoio.
Vai nella grotta e aiutalo a ricordarsi la ricetta del suo famoso tè 
alle erbe.

Per rendere la missione valida, devi essere nella grotta con Servillus. 
**e** il tuo ultimo comando prima di ``gsh check`` deve visualizzare la 
ricetta (incluso il titolo), ma nient'altro.

Comandi utili
=============

cat FILE
  Visualizza il contenuto del file.
  
head FILE
  Visualizza le prime 10 righe del file.
  
head -n K FILE
  Stampa le prime K righe del file.
  
Nota
----

Il "FILE" potrebbe contenere cartelle se il file in questione non è 
nella cartella corrente.
