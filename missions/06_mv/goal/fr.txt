Objectif
========

Ramassez toutes les pièces qui se trouvent dans la cave du château 
et placez les dans votre coffre (dans votre cabane, dans la forêt).


Commandes utiles
================

mv FICHIER1 FICHIER2 ... FICHIERn REPERTOIRE
  Déplace les fichiers vers le répertoire.
  Note : ``mv`` est une abréviation pour "move".

~
  Le symbole "~" est une abréviation pour le répertoire initial.
  Exemple : où que vous soyez, ``~/Taverne`` représente le dossier 
  (ou fichier) "Taverne" dans le répertoire initial.
