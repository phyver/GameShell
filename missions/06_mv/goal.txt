Objectif
========

ramasser les pièce trouvées dans la cave et les mettre 
dans le coffre (dans la cabane).


Commandes utilisées
===================

  mv FICHIER1 FICHIER2 ... FICHIERn DOSSIER
    déplace les fichiers dans un autre dossier

  ~
    abbréviation pour le répertoire initial (peut 
    dépendre du shell utilisé)
    exemple : ~/Dossier représente le dossier "Dossier" 
    dans le répertoire initial
