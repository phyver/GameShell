Objectif
========

Supprimez toutes les araignées qui grouillent dans la cave, sans 
toucher aux chauves-souris.


Motifs shell
============

?
  Le caractère "?" est un joker comme aux cartes, il remplace n'importe quel caractère.

*
  Le caractère "*" est un joker différent qui remplace n'importe quelle suite de 
  caractères (de zéro à beaucoup !). En programmation, une suite de caractère est 
  appelée une "chaîne".


On peut utiliser les jokers pour dénoter une liste de fichiers / 
répertoires du répertoire courant.

Par exemple, si le répertoire courant contient
      fichier-1 Rep-1 fichier-14 patate
alors
    *     -->  fichier-1 Rep-1 fichier-14 patate
    *1    -->  fichier-1 Rep-1
    *p*   -->  Rep-1 patate
    x*    -->  erreur, aucun fichier ne correspond
    *-?   -->  fichier-1 Rep-1
    *-??  -->  fichier-14
