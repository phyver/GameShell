Objectif
========

Ramassez toutes les pièces qui se trouvent dans le jardin en face du 
château et placez les dans votre coffre (dans votre hutte, dans la 
forêt).


Commandes utiles
================

mv FICHIER1 FICHIER2 ... FICHIERn REPERTOIRE
  Déplace les fichiers vers le répertoire.
  Note : ``mv`` est une abréviation pour "move".

~
  Le symbole "~" est une abréviation pour le répertoire initial.
  Par exemple, où que vous soyez, ``~/Taverne`` représente le 
  répertoire (ou fichier) "Taverne" dans le répertoire initial.
