Objectif
========

Combinez plusieurs commandes avec "|" pour retrouver le diamant dans 
labyrinthe et le déplacer dans votre coffre.

NOTE : le nom du fichier ne contient pas la chaine "diamant".
Il faut donc chercher un fichier qui contient la chaine "diamant".


Commandes utiles
================

find
  cherche des fichiers à partir du répertoire courant
  Les conditions peuvent porter sur le nom des fichiers, leur 
  taille, leur date de création, etc.

grep [OPTIONS] CHAINE FICHIER1 ... FICHIERn
  recherche une chaine parmi les fichiers
  Parmi les options, vous avez notamment
    -l  qui affiche juste le nom des fichiers contenant la chaine

xargs CMD
  lit des lignes sur l'entrée standard, et exécute la commande CMD 
  LIGNE1 LIGNE3 ... LIGNEn
  Cette commande est particulièrement utile derrière une redirection 
  "|"
