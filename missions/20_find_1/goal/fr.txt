Objectif
========

Retrouvez les pièces d'or dans le labyrinthe caché dans le jardin et 
déplacer la dans votre coffre.
Le jardin botanique se trouve devant le château.


Remarque : la description de la commande ``find`` est volontairement 
succincte. Vous devrez consulter l'aide pour comprendre comment 
l'utiliser.


Commandes utiles
================

find CONDITION
  cherche des fichiers à partir du répertoire courant

  Les conditions peuvent porter sur le nom des fichiers, leur 
  taille, leur date de création, etc.
  Deux conditions intéressantes sont :
    -name "MOTIF"
    -iname "MOTIF"

man CMD
  pour consulter l'aide de la commande CMD

  touches utilisées par la commande "man"
    q           quitter une page d'aide
    Space       faire défiler une page
    / CHAINE    rechercher une chaine
    n           se déplacer à la prochaine occurrence de la dernière
                chaine cherchée
