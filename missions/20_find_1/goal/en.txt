Mission goal
============

Find the gold coins in the maze hidden in the garden and move it to 
your chest.
The botanical garden can be found in front of the castle.


NOTE: the description of the ``find`` command is voluntarily very 
short. You will need to look at the manual to get more information.


Useful commands
===============

find CONDITION
  search for file starting from the current working directory

  Conditions restrict the files being searched, they can involve the 
  name, the size, the modification date, etc.
  Two interesting conditions are
    -name "GLOB"
    -iname "GLOB"

man CMD
  display the manual of the command

  important keybindings
    q           quit
    Space       scroll down one page
    / STRING    search for the string in the manual
    n           go to the next occurrence of the last search string
